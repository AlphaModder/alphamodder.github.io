space-email-api

  • on GitHub

space-email-api is a Rust client for the website Space Email, "an indirect communication platform that allows for a unique exchange of conversation over space and time." It is implemented with the reqwest library, and is fully compatible with Rust's new async-await features. space_email_scraper is a tool of mine built with space-email-api.
